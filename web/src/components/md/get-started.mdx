## GenQL

A SQL query generator from natural language.

### Get started

These are the ways to index your database schema

- Write the schema directly in JSON editor on the left pane.
- Upload the schema in `.json` file.
- Upload the schema in `.csv` file.

[Read here](/extract-schema) on how to extract schema in a CSV file. _(recommended)_

[This](/json-format.json) is the JSON structure of the database schema. [example](/schema-example.json).
